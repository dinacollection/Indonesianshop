## ✨ Features & Architecture Highlights

1. **2026 Modern Design Aesthetic**:
   - Deep Emerald (`#064e3b`) & Rich Gold Accent (`#d97706`) palette.
   - Glassmorphism Navigation Bar (`backdrop-filter: blur(16px)`).
   - Micro-animations, soft shadows, rounded UI components.

2. **300+ Realistic Local Indonesian Products**:
   - 18+ Comprehensive Categories: Frozen Food, Hijab & Fashion Muslim, Skincare BPOM, Cosmetics, Pakaian Wanita, Pakaian Pria, Sepatu & Sandal, Tas & Luggage, Dapur & Rumah, Elektronik, Snacks, Hampers, dll.
   - Rich specifications, Indonesian reviews, SKUs, and stock management.

3. **Live AI Chatbot Assistant**:
   - Floating widget with auto-reply knowledge base.
   - Instant answers on store location, vouchers, shipping fees, and product recommendations.

4. **Cart & Checkout System**:
   - Dynamic currency calculations in Rupiah (Rp).
   - Claimable Vouchers (`DHINA2026`, `ONGKIRFREE`, `MEMBERGOLDA`).
   - Payment choices: QRIS Instant, Virtual Account BCA/Mandiri, E-Wallet, and COD.
   - Instant printable transaction receipt generator.

5. **Visual Order Tracking**:
   - Live shipping step timeline (Order Placed -> Packing -> In Transit -> Delivered).

6. **SEO & PWA Ready**:
   - Semantic HTML5, JSON-LD Schema (Store & Products), `sitemap.xml`, `robots.txt`, and `manifest.json`.

## 🚀 GitHub Pages Deployment Guide

1. Upload or push all repository files directly to your GitHub repository root (`main` or `master` branch).
2. Go to **Settings > Pages**.
3. Under **Build and deployment**, select **Deploy from a branch**.
4. Select `main` / `master` branch and folder `/ (root)`.
5. Click **Save**. Your website will be live globally in less than 1 minute!
